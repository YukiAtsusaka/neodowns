# `neodowns`: Simulating Candidate Competition in Divided Societies

`neodowns` implements the computational model proposed in Atsusaka and Landsman (2024). When Does Ranked-Choice Voting Reduce Polariazation. Working Paper.
## Installation

`neodowns` can be installed using the following code:

``` r
remotes::install_github(
  "YukiAtsusaka/neodowns",
  dependencies = TRUE
)
```
